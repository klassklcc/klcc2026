Round 2 Part 3 - Array Split

Finally, you make it past the robots and end up at another locked stairwell. However, this time you are given a long array, and you have to split the array into two such that the difference between their sums are minimised

Example Case:

Input: [7, 2, 3, 10, 5, 1]
Output: 4
Explanation: We can split it as [7, 2, 3] and [10, 5, 1] which makes their sums 12 and 16. |12-16| = 4

Input : An array of integers. However, the input is too long to display. Access it here
Output : Output the absolute difference between the two sums.
