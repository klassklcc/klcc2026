Debugging Questions

Can you find all of the issues in the code? Good luck!
The code for the following questions will be in Python. If you need another language such as Java or C++, please let us know.



Question 1

What is the issue with the following code? It should sum all the numbers between 1 and 10 inclusive, but it does not work. 

      total = 0
      for i in range(1,10):
        total += i


Question 2

What is the issue with the following code? It should multiply the input by two, but it does not work. 

      age = input()
      print(2*age)


Question 3

What is the issue with the following code? The code is trying to determine if a number is strictly bigger than everything else in the list, but it prints True instead of false. 

      testlist = [1, 6, 7, 4, 5, 2]
      number = 3
      for i in range(len(testlist)):
        bigger = True
        if number <= testlist[i]:
          bigger = False
        else:
          bigger = True
      print(bigger)


Question 4

What is the issue with the following code? It is trying to determine if a list is sorted but it raises an error. 

      list1 = [1, 6, 7, 4, 5, 2]
      sort = True
      for i in range(len(list1)):
        if list1[i] > list1[i+1]:
          sort = False


Question 5

What is the issue with the following code? It should print the sorted list, but it does not work. 

      list1 = [1, 6, 7, 4, 5, 2]
      print(list1.sort())
